ï»¿ The following documentation is for the node JavaScript server that runs the game "An Edge of 5 Kingdoms" submitted as part of the coursework for the Mobile Game Development module. The server has 2 main parts a server object and a set of api objects which can be broken down into a score api for highscore tracking and a news api for streaming news updates about the game to the main menu.

Score API

The score API has the following requests

POST

- PostScore

PostScore is a simple post request which expects a list of arguments in the request to post a new score. Like so...

{ "name":"William", "highscore": "100", "service": "VERSION1" }

The new score will be added into the database and will show up at the end of the game providing they are running VERSION1 of the game. As highscores are seperated for each version

- DeleteScore

DeleteScore is a simple post request which expects some arguments in the body of the request which identify the element to delete.

{ "name":"William", "highscore": "100", "service":"VERSION1" }

The element in the database which matches these values will be deleted.

GET

- GetScores

Get score is a simple get request which will return a sorted list of high scores. The list length will be 5 as that is the limit. The array will contain a JSON object at each index which will represent a high score entry which will consist of...

  • Name
  • Highscore
  • Service

The array can be iterated and then used to popular a leaderboard table.

News API

The news API has the following request options

POST

- PostNews

PostNews is a simple post request which expects a list or arguments in the request to post a new score. Like so...

{ "Heading":"Header", "Service": "VERSION1", "Text": "Text", }

A news story will be added to the database which will then be displayed in the app in the most recent order.

  • DeleteNews

DeleteNews is a simple post request which expects some arguments in the body of the request which identify the element to delete.

{ "postID":"0", "service": "VERSION1", }

The element in the database which matches these values will be deleted.

GET

- GetNews

Get score is a simple get request which will return a sorted list of news story's (Sorted by most recent). The current number will be 5 as that is the limit. The array will contain a JSON object at each index which will represent a news entry which will consist of...

  • Heading
  • Text
  • Service
  • PostID

The array can be iterated and then used to popular a news table.
